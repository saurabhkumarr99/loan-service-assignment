# Loan API Integration Service

## 📌 Overview
This is a Spring Boot REST API that integrates with an external loan API, processes EMI data, and exposes a clean endpoint to fetch loan details.

---

## 🚀 Features
- Integrates with external loan API
- Extracts due EMI details
- Returns structured response
- Clean layered architecture (Controller → Service → Integration)
- Logging enabled
- Generic API response wrapper

---

## 🛠 Tech Stack
- Java 17
- Spring Boot
- REST API
- Lombok
- SLF4J Logging
